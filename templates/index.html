About Project
Face Recognition Based Attendance System

All students of the class must register by entering the necessary information. The images of the student are then captured and stored in a dataset. During each session, faces are recognized from the live streaming video of the classroom. The detected faces are compared with the images in the dataset. If a match is found, attendance is determined for each student. The task of the proposed system is to capture the face of each student and store it in a database for attendance. The student's face should be captured in such a way that all the features of the student's face should be recognized, and the student's seat and posture should also be recognized. Teachers do not need to attend the class manually as the system captures the video and through post-processing steps recognizes faces and updates the attendance database.

Purpose and objective


The purpose of this project is to create a facial recognition attendance system. The expected results to achieve the goals are:
● Recognize face parts from video frames.
● Extract useful features from detected faces.
● Classification of features to recognize faces.
● Attendance registration of identified students.
